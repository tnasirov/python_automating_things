This fixlet verifies the access permissions of /var/log/wtmp. The default permissions do not allow write access for world.<br /> The access permissions of the /var/log/wtmp file will be compared with the value you specify as the desired value for the 'OSR Permissions for /var/log/wtmp' parameter. <br /><br />If the setting allows group write access, this fixlet will be relevant if the file has group write access and the group does not meet one of the following conditions:<ul><li>The associated group is used only by set-GID operating system programs to avoid a need for root only access privileges.<em> Note: The fixlet does not check this condition.</em></li><li>The associated group has GID&lt;= 99</li><li>The client system is Red Hat version 6 and the associated group has GID&lt;= 99 or &gt;=101 and &lt;=199</li></ul>
